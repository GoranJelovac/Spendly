Kategorije su sekcije koje se otvaraju/zatvaraju. Svaka kategorija prikazuje ukupan iznos i broj linija. Klik otvara listu linija ispod.

B: Grouped Table

Tabela ostaje, ali je grupisana po kategorijama. Kategorija je header red koji se kolapsira. Zadržava sve kolone (Name, Code, Type, Amount).
